# Secure Banking System

A **C++ console-based banking system** demonstrating object-oriented programming, file handling, and bitwise permissions. This project simulates real-world banking operations like deposits, withdrawals, and account management.

---

## Features

- **Two Account Types**:
  - Savings Account (with interest rate)
  - Current Account (with overdraft facility)
- **Transaction Management**:
  - Deposit and withdraw funds
  - Transaction history tracking
- **VIP Accounts**:
  - Special VIP status with permissions
- **Monthly Summary**:
  - Automatic tracking of monthly deposits and withdrawals
- **File Handling**:
  - Save and load account data from a file
- **Permissions**:
  - Withdraw, Deposit, Transfer using bitwise flags
- **Polymorphism & Inheritance**:
  - Abstract base class `Account`
  - Derived classes `SavingsAccount` and `CurrentAccount`
